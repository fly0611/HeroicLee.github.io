转义输出类型

转义类型 解释
d,i 十进制
o 八进制
u 十进制
x 十六进制(小写)
X 十六进制(小写)
e 科学计数法浮点数(小写)
E 科学计数法浮点数(大写)
f,F 十进制浮点数
C 接收整数(会转换为对应ascii码字符)、 单字符)
s 字符串

输入input

  • Python2： raw_input() input()
    • input 会默认用户输入的是合法的Python表达式
    • raw_input 会把所有的输入当做原始数据进行处理
  • Python3： input()
    • input 会把所有的输入当做原始数据进行处理
